
# ATTENFAI

AI-powered employee attendance and productivity monitoring system.

## Features
- Automatic login & logout detection
- Productivity scoring using AI
- Offline-first desktop monitoring
- Secure JWT authentication
- Manager dashboard & analytics

## Tech Stack
- Desktop: PyQt + Python
- Backend: FastAPI
- Web: React
- AI: Scikit-learn, PyTorch
- Database: SQLite (Local), RDS, MongoDB, Redis
- Deployment: Docker
